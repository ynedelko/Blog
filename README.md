# _BLOG_

##### Personal Blog created with Ruby on Rails

#### By _**Yelena Nedelko**_

## Live App deployed on Heroku:
* _http://yelenablog.herokuapp.com/_

## Description

_User can:_
* _Add, update, delete and see a list of my posts._
* _Add and delete a tag for a specific post._
* _View post in order by last modified._
* _View comments for posts._
_Logged-in user can:_
* _Add, edit, and delete comments._
* _See author names for comments._


## Technology Used
* _Ruby_
* _Rails_
* _RSpec_
* _Capybara_
* _Postgres_
* _Rake_
* _Active Record_
* _Shoulda-matchers_
* _Acts-as-taggable-on_
* _Devise_
* _Factory_Girl_
* _Simple_Cov_
* _Rails_12Factor_

## Database Setup/ Installation

_In rake:_<br>
_db:migrate_<br>
_all database tables listed in schema.rb_
<br>

_Webserver testing:_
* _Install required gems_
* _$ bundle install_
* _rake db:test:prepare_
* _$ rails server_
* _Navigate to localhost:3000_
